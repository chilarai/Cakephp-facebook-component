Cakephp-facebook-component by chilarai

A cakephp facebook component with easy GUI installer with live examples to connect to apps, send app requests, write on wall, view profile picture, friends list, and many more

Dependencies

php-curl extensions

Install

Please use the cakephp-facebook auto installer if you want to setup manually then read the 'readme' file
